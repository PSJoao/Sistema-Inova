# Campo `data_despache` na Expedição

Adicionar o campo `data_despache` (DATE) à `cached_etiquetas_ml`, capturando a data de despacho a partir das etiquetas PDF e ZPL, exibindo na dashboard e permitindo filtro por período.

## Proposta de Mudanças

### 1. Banco de Dados — Migração

Adicionar coluna `data_despache DATE` à tabela `cached_etiquetas_ml`:

```sql
ALTER TABLE cached_etiquetas_ml ADD COLUMN IF NOT EXISTS data_despache DATE;
```

---

### 2. Parsing da Data de Despacho

Função utilitária `parseDataDespache(textoDespacho, anoReferencia)` que converte textos como:
- PDF: `"sábado 9/mai"` → `2026-05-09`
- ZPL: `"s_c3_a1bado 4/abr"` (hex encoded) → `2026-04-04`

Mapeamento de meses abreviados em PT-BR: `jan→1, fev→2, mar→3, abr→4, mai→5, jun→6, jul→7, ago→8, set→9, out→10, nov→11, dez→12`

O `anoReferencia` será `new Date().getFullYear()` (ano atual no momento do processamento).
